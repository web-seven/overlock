First post-1.0 patch release. Brings official documentation tooling, fixes Docker Desktop on Linux setup, and restores the install script after the project rename.

<!-- truncate -->

## What's new

- **Docusaurus documentation site** ([#450](https://github.com/web-seven/overlock/pull/450)) — `./docs` is now a Docusaurus project with sidebar, search, and edit-on-GitHub links. The same Markdown still works on GitHub.
- **GitHub-style admonitions in docs** ([#467](https://github.com/web-seven/overlock/pull/467)) — `> [!NOTE]` / `[!WARNING]` syntax renders as proper Docusaurus admonitions.
- **Docker Desktop on Linux setup guide** ([#465](https://github.com/web-seven/overlock/pull/465)).

## Bug fixes

- **Docker Desktop compatibility and remote `wg0` reuse** ([#464](https://github.com/web-seven/overlock/pull/464)) — environments now come up cleanly on Docker Desktop, and existing WireGuard interfaces on remote nodes are reused instead of failing.
- **Install script repo URL** ([#458](https://github.com/web-seven/overlock/pull/458)) — `install.sh` now points at `web-seven/overlock` after the org rename.
- **Fix Documentation deploy workflow on `main`** ([#462](https://github.com/web-seven/overlock/pull/462)).
