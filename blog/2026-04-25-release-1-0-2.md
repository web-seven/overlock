A small but important patch release that unblocks pulling container images from clusters running behind PPPoE links.

<!-- truncate -->

## Bug fixes

- **Fix image pulls behind PPPoE links** ([#468](https://github.com/web-seven/overlock/pull/468)) — pod MTU and TCP MSS are now clamped so the registry handshake completes on connections with a smaller MTU than standard Ethernet. If you saw image pulls hanging on home/PPPoE networks, this release is for you.
